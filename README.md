# SBTI 人格测试

一个趣味性人格测试应用，通过分析用户的回答来匹配26种不同的人格类型。

![SBTI](https://sbti.oucloud.top/image/CTRL.png)

## 📋 项目简介

SBTI是一个创新的在线人格测试工具，通过30道精心设计的问题，分析用户的15个心理维度，最终匹配到26种独特人格类型之一。测试仅需5-10分钟，帮助用户了解自己的性格特点、行为模式和心理特征。

与传统的MBTI测试不同，SBTI更注重现代人的实际心理特征和行为模式，测试内容更加贴近当代人的生活场景和心理状态。

## ✨ 功能特性

- **30道趣味测试题** - 涵盖自我、情感、态度、行动、社交等5大模型
- **26种独特人格类型** - 从CTRL拿捏者到BOSS领导者，每种类型都有独特描述
- **15个心理维度分析** - 全面分析用户的心理特征
- **智能匹配系统** - 基于用户答案精准匹配人格类型
- **详细的测试结果** - 提供人格类型描述、维度评分和发展建议
- **FAQ问答系统** - 12个常见问题，帮助用户了解测试
- **社交分享功能** - 支持微信、微博、QQ、QQ空间、豆瓣分享
- **结果下载功能** - 一键下载测试结果为HTML文件
- **完全免费使用** - 无需注册，随时测试

每个人格类型都有其独特的心理特征、行为模式和人格描述。

## 📝 使用说明

### 开始测试
1. 点击"开始测试"按钮
2. 依次回答30道问题
3. 提交并查看结果

### 查看结果
- 人格类型介绍
- 心理维度评分
- 匹配度分析
- 发展建议

### 分享结果
- 使用分享按钮分享到社交平台
- 或使用下载功能保存结果
